# 🎬 MERN YouTube Clone

A full-stack YouTube Clone built using the **MERN Stack (MongoDB, Express, React, Node.js)**.  
This project allows users to watch videos, upload videos, create channels, and interact with content.

---

## 🚀 Features

### 👤 Authentication
- User Registration
- User Login
- JWT Authentication

### 🎥 Videos
- Upload Video
- Edit Video
- Delete Video
- Watch Videos
- Like / Dislike Videos

### 💬 Comments
- Add Comments
- View Comments
- Delete Comments

### 📺 Channel
- Create Channel
- View Channel Page
- Manage Channel Videos

### 🔍 Search & Filters
- Search videos by title
- Filter videos by category

### 🎨 UI
- Responsive Design
- Video Grid Layout
- Sidebar Navigation
- Header Search Bar

---

## 🛠 Tech Stack

### Frontend
- React
- React Router
- Axios
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JSON Web Token (JWT)

---

## ⚙️ Installation

### Clone the repository
git remote add origin https://github.com/adarshanit1125/youtube-clone.git


### Backend Setup
cd backend
npm install
npm run dev


### Frontend Setup
cd frontend
npm install
npm run dev


---

## 🌐 API Endpoints

### Auth
POST /api/auth/register
POST /api/auth/login

### Videos
GET /api/videos
GET /api/videos/:id
POST /api/videos
PUT /api/videos/:id
DELETE /api/videos/:id
